# Contributing

Thank you for your interest in Think Like SQL.

This project is an open educational initiative focused on helping learners understand SQL through reasoning rather than memorization.

Contributions are welcome in the following areas:

- Typographical corrections
- Clarifications and explanations
- SQL examples
- Practice exercises
- Diagram improvements
- Suggestions for additional case studies

Please ensure that any contributions maintain the educational philosophy of the project:
explain *why* SQL behaves the way it does before introducing syntax or implementation details.
