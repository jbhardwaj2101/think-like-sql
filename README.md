# Think Like SQL

**From Writing Queries to Designing Efficient Solutions**

A free practical SQL learning project for analysts, analytics engineers, BI developers, and aspiring data engineers.

## Purpose

Most SQL resources teach syntax. This project focuses on how SQL works, how databases interpret queries, and how to reason through correctness, readability, and performance.

## Project Structure

- `book/` — Chapter drafts and final book content
- `linkedin-posts/` — LinkedIn post drafts based on each chapter
- `sql/` — Sample data, exercises, and solutions
- `diagrams/` — Visual explanations and diagrams
